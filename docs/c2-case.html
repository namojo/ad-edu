C2 완성형 사례 — 대행사 "주간 캠페인 성과 리포트" 자동화

대행사가 매주 반복하는 캠페인 성과 리포트를 automation-harness로 자동화한 완성형 사례입니다. 업무 분해 → 패턴 선택 → 4종 문서 패키지 → 매주 재실행 → 실제처럼 집필한 샘플 리포트 전문까지 담았습니다.

⚠️ 이 문서의 브랜드("제로톡")와 모든 수치는 학습용 예시 데이터입니다.

1. 상황 — 무엇을 자동화했나

대행사 A의 퍼포먼스팀은 담당 브랜드 **제로톡(제로 슈거 스파클링)**의 여름 캠페인을 운영 중입니다. 매주 월요일, 담당 AE는 지난주 성과를 정리해 내부 보고와 광고주 발표용 자료를 만듭니다.

기존에는 브랜드 하나당 23시간이 걸렸습니다. 매체 대시보드에서 숫자를 긁고, 엑셀 비교표를 만들고, 코멘트를 달고, 이걸 발표 덱과 보고 문서로 각각 옮기는 반복 작업이었죠. 이 업무를 하네스로 한 번 설계해, 이제는 **데이터만 넣으면 2030분 만에 4종 문서**가 나옵니다.

2. 업무 분해 — 역할 나누기

큰 일("주간 리포트 만들기")을 세 조각으로 나눠 각 담당에게 맡겼습니다. 이것이 계층적 위임입니다.

역할 담당(팀원) 구체적으로 하는 일 입력 → 출력
데이터 요약 데이터 담당(리서처) 매체별 성과를 정리하고 지난주와 비교. 증감을 표로 원시 성과 데이터 → 비교표 + 증감
인사이트 도출 전략 담당(인사이트 에디터) "왜 이런 결과인가"를 근거와 함께 해석 비교표 → 인사이트 3개(주장→근거)
액션 제안 기획 담당(인사이트 에디터/AE) 다음 주 무엇을 바꿀지 실행안 인사이트 → 액션 3개(무엇을·왜·기대효과)
문서 마감 문서 프로듀서(doc-producer) 위 결과를 4종 문서로 자동 생성 세 결과 → PPT·Word·Excel·PDF

각 조각은 앞사람의 결과를 뒷사람이 받습니다(데이터 → 인사이트 → 액션). 순서 협업이면서, 동시에 각자 다른 전문가가 맡는 구조입니다.

3. 패턴 선택 근거 — 왜 계층적 위임 + 전문가 풀인가

이 업무에 두 패턴을 함께 쓴 이유는 분명합니다.

계층적 위임을 쓴 이유: "주간 리포트"는 그 자체로는 너무 큰 덩어리입니다. 데이터 정리, 해석, 실행안은 성격이 다른 일이라 한 사람이 다 하면 품질이 고르지 않습니다. 큰 일을 작은 일로 쪼개 아래로 맡기면 각 조각의 품질이 올라가고, 어디서 문제가 났는지도 쉽게 짚힙니다. 다음 주에 "인사이트만 다시 뽑아 줘"처럼 부분 재실행도 가능해집니다.

전문가 풀을 쓴 이유: 리포트 업무엔 카피라이터나 아트디렉터가 필요 없습니다. 매번 전원을 부르는 대신 이 업무에 맞는 데이터·전략·기획·문서 담당만 골라 부릅니다. 빠르고, 결과도 군더더기가 없습니다. 만약 다음 달에 "경쟁사 광고 소재 분석"이 필요해지면, 그때 리서처를 추가로 풀에서 꺼내 붙이면 됩니다.

정리: 계층적 위임으로 일을 나누고(품질·부분 재실행), 전문가 풀로 사람을 고른다(속도·확장성). 반복 리포트 업무엔 이 조합이 가장 잘 맞습니다.

4. 자동 생성 문서 패키지 — 무엇이 들어가나

한 번의 요청으로 아래 4종이 한꺼번에 나옵니다. 각 문서는 용도가 다릅니다.

4-1. 발표용 PPT 덱 (pptx) — 광고주 보고용, 요약 중심

빠르게 훑고 결정할 수 있게 핵심만 담습니다.

슬라이드 내용
1. 표지 브랜드·캠페인명·주차(예: 제로톡 여름 캠페인 / 7월 1주차)
2. 한 장 요약 이번 주 핵심 3줄 + 총 전환·전환당 비용 큰 숫자
3. 매체별 성과 매체 4종 비교 막대 + 전주 대비 화살표
4. 인사이트 잘된 점·아쉬운 점 각 1~2개
5. 다음 주 액션 액션 3개 카드 (무엇을·기대효과)
6. 부록 상세 지표는 Excel 참조 안내

4-2. 상세 Word 보고서 (docx) — 내부 기록용, 전체 내용

발표 덱이 못 담는 맥락과 근거를 모두 기록합니다.

[상세 보고서 목차]
1. 캠페인 개요 (기간·목표·예산 집행률)
2. 종합 요약
3. 매체별 성과 상세
   3-1. 메타  3-2. 유튜브  3-3. 검색  3-4. 틱톡
4. 전주 대비 변화 분석
5. 인사이트 (주장 → 근거 → 시사점)
6. 다음 주 실행 계획 (액션별 담당·기한·기대효과)
7. 리스크 및 특이사항
8. 부록: 데이터 출처·집계 기준

4-3. 성과 지표 Excel 시트 (xlsx) — 데이터 원본, 재가공용

숫자를 다시 계산하거나 다른 표로 만들 때 쓰는 원천 데이터입니다.

열 이름 설명
매체 메타 / 유튜브 / 검색 / 틱톡
노출 광고 노출 수
클릭 클릭 수
CTR(%) 클릭률 = 클릭 / 노출
전환 전환 수
CVR(%) 전환율 = 전환 / 클릭
비용 집행 비용(원)
CPA(원) 전환당 비용 = 비용 / 전환
전주 전환 지난주 전환 수
전주 대비 증감률(%) (이번주-지난주)/지난주

4-4단계 피드백으로 추가된 '전주 대비 증감률(%)' 열이 여기 반영되어 있습니다(3장·변경 이력 참조).

4-4. 통합 PDF (pdf) — 배포·인쇄용, 한 파일로

위 덱·보고서·시트 핵심을 하나로 합친 배포본입니다. 이메일 첨부나 인쇄 회람에 씁니다. 표지 → 요약 → 지표표 → 인사이트 → 액션 순서로 흐릅니다.

5. 매주 재실행 방법 — 하네스 진화와 변경 이력

5-1. 매주 하는 일은 딱 하나

하네스를 한 번 만든 뒤로는, 매주 빈 데이터 양식에 이번 주 숫자만 채워 넣고 실행합니다.

이번 주 데이터로 리포트 만들어줘. (아래 양식에 채워서 전달)
- 기간: [YYYY 몇월 몇째 주]
- 매체별 성과: 메타 / 유튜브 / 검색 / 틱톡 (노출·클릭·전환·비용)
지난주와 같은 틀로 4종 문서까지 만들어줘.

"지난주와 같은 틀로"가 포인트입니다. 이 한마디로 매주 동일한 포맷이 보장됩니다.

5-2. 진화 — 피드백이 쌓이면 하네스가 좋아진다

쓰다 보면 개선점이 나옵니다. 반영할 때마다 하네스가 조금씩 진화하고, 변경 이력에 한 줄을 남깁니다.

날짜 변경 내용 사유
2026-07-07 주간 리포트 하네스 초기 구성 반복 업무 자동화 시작
2026-07-14 인사이트 코멘트 2문장 이내로 축약 광고주가 짧은 요약 선호
2026-07-14 Excel에 '전주 대비 증감률(%)' 열 추가 증감 한눈에 파악 필요
2026-07-21 덱 표지에 캠페인 진행률(%) 추가 예산 소진 속도 공유 요청

이력이 쌓이면 "왜 이렇게 바꿨더라?"를 되짚을 수 있고, 담당자가 바뀌어도 인수인계가 됩니다.

5-3. 운영 점검(감사)

분기에 한 번, 혹은 결과가 이상할 때 "이 하네스 구성 점검해 줘"로 어긋남(드리프트)을 확인합니다. 예: 정의는 "경쟁사 3곳"인데 실제로 5곳이 들어가고 있진 않은지, 옛 템플릿으로 나오고 있진 않은지.

검증은 매주. 자동화됐다고 눈을 떼면 안 됩니다. 특히 Excel 합계, 전주 대비 값, 액션의 실행 가능성은 마감 전 반드시 확인하세요. AI는 매번 조금씩 다르게 답하므로, 지난주와 표현이 달라지거나 숫자를 잘못 옮기는 일이 드물게 생깁니다. 사람의 눈이 마지막 관문입니다.

6. 샘플 주간 리포트 (전문) — 예시 데이터

아래는 하네스가 실제로 뽑아낸 것과 같은 형태의 주간 리포트 본문입니다. 모든 수치는 학습용 예시 데이터입니다.

📊 제로톡 여름 캠페인 · 7월 1주차 주간 리포트

기간: 2026-07-01 ~ 07-07 · 집행 예산: 8,750만원 (누적 소진율 41%) · 주 KPI: 전환(제품 구매 페이지 도달)

① 종합 요약

이번 주 총 전환 1,240건으로 전주(1,051건) 대비 +18% 늘었고, 전환당 비용(CPA)은 8,900원으로 전주(10,110원)보다 -12% 개선됐습니다. 유튜브 조회 기반 캠페인이 전환 효율을 크게 끌어올린 것이 주효했습니다. 반면 틱톡은 노출이 늘었으나 전환은 정체됐고, 검색은 브랜드 키워드 유입이 안정적으로 유지됐습니다.

② 매체별 성과 (예시 데이터)

매체 노출 클릭 CTR 전환 CVR 비용(원) CPA(원) 전주 대비 전환
메타 1,820,000 41,900 2.30% 430 1.03% 3,870,000 9,000 +9%
유튜브 2,450,000 58,800 2.40% 520 0.88% 4,160,000 8,000 +34%
검색 96,000 12,500 13.0% 210 1.68% 1,680,000 8,000 +5%
틱톡 1,510,000 30,200 2.00% 80 0.26% 1,320,000 16,500 -3%
합계 5,876,000 143,400 2.44% 1,240 0.86% 11,030,000 8,900 +18%

③ 인사이트 3개 (주장 → 근거)

  1. 유튜브가 이번 주 성장을 견인했다. 유튜브 전환이 전주 대비 +34%로 가장 크게 늘었고 CPA도 8,000원으로 낮습니다. 조회 후 검색 유입으로 이어지는 흐름이 관측됩니다.
  2. 틱톡은 노출은 사지만 전환으로 못 잇고 있다. 틱톡 CVR 0.26%는 다른 매체의 1/3~1/4 수준이고 CPA는 16,500원으로 가장 비쌉니다. 소재가 상단 노출은 잡되 구매 동기까지 밀어주지 못하는 것으로 보입니다.
  3. 검색은 낭비 클릭이 섞여 효율을 갉아먹고 있다. 검색 CTR은 13%로 높지만, 브랜드 무관 일반 키워드에서 전환 없는 클릭이 확인됩니다. 제외 키워드 정비 여지가 있습니다.

④ 다음 주 액션 3개

# 액션 왜 기대 효과
1 유튜브 예산 15% 증액 전환 성장·낮은 CPA로 확장 여력 확인 전환 +80건 예상, 전체 CPA 추가 하락
2 틱톡 소재 3종 교체 A/B 현 소재의 전환 전환력 부족 CVR 0.26%→0.5%대 회복 목표
3 검색 제외 키워드 정비 전환 없는 일반 키워드 낭비 클릭 검색 CPA 8,000원→7,000원대, 예산 재분배

본 리포트는 automation-harness가 데이터 요약·인사이트·액션 담당과 문서 프로듀서를 조합해 자동 생성한 예시입니다. 수치는 모두 학습용 가상 데이터입니다.

7. 이 사례에서 가져갈 것

  • 한 번 설계, 매주 재사용. 업무를 세 조각으로 나눠 맡기고 문서 프로듀서에게 마감을 시키는 구조를 한 번 만들면, 이후엔 데이터만 넣습니다.
  • 패턴은 이유가 있어야 한다. 계층적 위임(품질·부분 재실행)과 전문가 풀(속도·확장)을 왜 골랐는지 설명할 수 있어야 합니다.
  • 문서는 용도별로. 덱(요약)·Word(기록)·Excel(원본)·PDF(배포)는 목적이 다릅니다.
  • 진화와 검증을 습관으로. 고칠 때마다 변경 이력 한 줄, 마감 전 숫자 검증 한 번.
