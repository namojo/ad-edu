# 시장 조사 (Market Research)

광고 기획의 첫 단추. 감이 아니라 **근거**로 판단하도록, 시장·소비자·경쟁사·트렌드를 조사해 출처가 붙은 사실을 모은다.

## 왜 이 매뉴얼인가

조사가 실무에서 무너지는 이유는 딱 셋이다. (1) 범위를 안 정해서 조사가 끝없이 번진다. (2) "요즘 이게 대세래요" 처럼 출처 없는 인상을 사실처럼 적는다. (3) 서로 다른 자료가 충돌할 때 한쪽만 골라 적어 편향된다. 이 매뉴얼은 이 셋을 막는 절차다. 조사는 **재현 가능해야** 신뢰받는다. 즉 누가 봐도 "어디서 나온 이야기인지" 추적되어야 한다.

## 워크플로우

### 1단계 — 조사 범위 고정 (가장 중요)

조사를 시작하기 전에 아래 3축을 **먼저 문장으로 확정**한다. 이걸 안 하면 조사가 산으로 간다.

- **카테고리**: 무엇을 조사하는가. (예: "제로 슈거 탄산음료", "20대 대상 배달앱"). 너무 넓으면("음료 시장") 한 단계 좁힌다.
- **기간**: 언제 자료인가. (예: "최근 2년", "2025~2026"). 오래된 트렌드를 최신처럼 쓰지 않기 위함이다.
- **지역**: 어느 시장인가. (예: "국내", "북미"). 해외 사례를 국내처럼 인용하는 실수를 막는다.

범위가 지시에 없으면 **가장 합리적인 기본값을 스스로 정하고, 산출물 맨 위에 명시**한다. 예: "범위: 국내 / 제로 슈거 음료 / 최근 2년." 추측으로 좁혀도 좋으니 반드시 밝힌다. 범위를 밝히면 나중에 "지역만 바꿔서 다시" 같은 재조사가 쉬워진다.

### 2단계 — 4개 조사 축 나눠 파기

조사는 아래 4개 축으로 나눠서 각각 별도로 판다. 섞으면 빠지는 축이 생긴다.

| 축 | 무엇을 찾나 | 핵심 질문 |
|---|---|---|
| 시장 | 규모·성장·구조 | 시장이 크는가 줄어드는가, 얼마나 되나 |
| 소비자 | 니즈·불만·구매 이유 | 왜 사고, 왜 안 사나. 무엇에 불만인가 |
| 경쟁 | 주요 플레이어·포지셔닝 | 누가 있고, 각자 뭘로 파나. 빈틈은 어디인가 |
| 트렌드 | 최근 변화·신호 | 요즘 뜨는 것, 지는 것. 근거는 무엇인가 |

각 축마다 **최소 2개 이상의 서로 다른 출처**를 확인한다. 하나만 보면 그 출처의 편향을 그대로 물려받는다.

### 3단계 — 근거·출처 명기

발견 하나하나에 출처를 붙인다. 출처 없는 문장은 조사가 아니라 의견이다.

- 수치·통계에는 **출처명 + 시점**을 붙인다. (예: "국내 제로 음료 시장 약 1.2조 원(○○리서치, 2025)")
- 원문 링크나 자료명이 있으면 함께 남긴다.
- 직접 확인 못 하고 추정한 값은 **"추정"이라고 반드시 표기**한다. 그럴듯한 숫자를 사실처럼 적지 않는다.

> 웹 조사가 필요하고 깊게 파야 하면 `deep-research` 스킬(여러 출처를 교차 검증하는 심화 조사 도구)을 함께 쓴다.

### 4단계 — 상충 정보 처리

두 자료가 서로 다른 이야기를 하면, **한쪽을 고르지 말고 둘 다 적는다.** 이게 이 매뉴얼의 핵심 원칙이다.

```
[상충] 시장 성장률
- A자료(○○리서치, 2025): 연 12% 성장
- B자료(△△경제, 2026): 연 6% 성장
→ 판단: 집계 기준이 다를 가능성. 캠페인 판단 시 보수적으로 6% 기준 권장.
```

왜냐하면 상충을 감추면, 그 위에 쌓는 모든 기획이 모래성이 되기 때문이다. 충돌을 드러내는 것이 오히려 신뢰를 높인다.

## 산출물 형식

```markdown
# [카테고리] 시장 조사
> 범위: [지역] / [카테고리] / [기간]   ← 반드시 맨 위에

## 1. 시장
- 발견 (출처, 시점)

## 2. 소비자
- 발견 (출처, 시점)

## 3. 경쟁
- 주요 플레이어와 포지셔닝 (출처)

## 4. 트렌드
- 최근 신호 (출처, 시점)

## 상충·유의 사항
- [상충] 항목: A(출처) vs B(출처) → 판단

## 조사 요약 (3~5줄)
```

## 품질 기준

- [ ] 산출물 맨 위에 범위(카테고리·기간·지역) 명시
- [ ] 모든 수치·주장에 출처 + 시점
- [ ] 각 축마다 출처 2개 이상 교차 확인
- [ ] 추정값은 "추정" 표기
- [ ] 상충 정보는 양쪽 출처 병기
- [ ] 조사 요약이 발견과 일치 (요약에서 없던 이야기 만들지 않기)

## 자주 하는 실수

- **범위를 안 정하고 시작** → 조사가 끝없이 번진다. 반드시 1단계부터.
- **출처 없이 "요즘 트렌드는…"** → 인상을 사실로 적은 것. 근거를 못 찾으면 "추정" 또는 삭제.
- **해외 자료를 국내처럼 인용** → 지역 축을 고정하지 않아 생기는 실수.
- **오래된 자료를 최신처럼** → 시점 표기로 방지.
- **상충을 숨기고 한쪽만** → 편향된 조사. 둘 다 적는다.
- **조사와 해석을 섞기** → 여기서는 사실만 모은다. "그래서 어떻게 하자"는 insight-synthesis 의 몫.

## 재호출 지침

- "조사 다시" → 이전 범위를 유지하고 출처를 새로 보강한다.
- "범위 좁혀서/바꿔서 재조사" → 1단계 범위만 바꾸고 4축을 다시 판다.
- "출처 보완" → 출처 없는 문장을 찾아 근거를 채우거나 "추정"으로 낮춘다.
- "○○축만 수정" → 해당 축만 다시 파고 나머지는 보존한다.

> AI는 같은 질문에도 매번 조금씩 다르게 답한다. 조사 결과를 그대로 믿지 말고, 핵심 수치는 한 번은 직접 의심하고 출처를 확인한다.
