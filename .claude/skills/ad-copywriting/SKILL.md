# 광고 카피 작성 (Ad Copywriting)

인사이트를 사람 마음을 움직이는 **한 문장**으로 바꾸는 일. 하나의 정답이 아니라 여러 앵글의 후보를 만들어 고를 수 있게 한다.

## 왜 이 매뉴얼인가

카피가 약한 이유는 대개 (1) 한 방향으로만 써서 더 좋은 각도를 놓치거나, (2) 제품 설명만 하고 마음을 안 건드리거나, (3) 멋있지만 브랜드 톤과 안 맞아서다. 이 매뉴얼은 **3앵글로 발산 → 형식별로 정리 → 톤 체크 → A/B 변형** 순으로 이걸 막는다. 카피는 예술이 아니라 **선택지를 만드는 일**이다. 좋은 카피 한 줄보다, 비교할 수 있는 여러 줄이 실무에서 강하다.

## 시작 전 확보할 것

- **무엇을 팔며 핵심 강점은 무엇인가** (제품/서비스)
- **누구에게** (타깃)
- **어떤 인사이트를 담을 것인가** (insight-synthesis 산출물이 있으면 최우선 재료)
- **브랜드 톤앤매너** — 밝은가 진중한가, 반말인가 존댓말인가, 써도 되는 말/금지어. 없으면 합리적으로 가정하고 산출물에 명시한다.

## 워크플로우

### 1단계 — 3앵글로 발산

같은 메시지를 **세 가지 방향**으로 쓴다. 한 앵글만 쓰면 더 좋은 각도를 못 본다.

| 앵글 | 무엇을 자극하나 | 언제 강한가 |
|---|---|---|
| 감성 | 기분·욕망·공감·동경 | 브랜드 이미지, 라이프스타일 소구 |
| 기능 | 강점·효능·차별점·근거 | 스펙이 확실한 제품, 비교 우위 |
| 유머 | 재미·의외성·위트 | 젊은 타깃, 바이럴, 가벼운 톤 |

각 앵글마다 최소 2~3개 후보를 뽑는다. 유머 앵글은 **선을 넘지 않는지**(민감·비하·오해 소지) 반드시 자기검열한다.

### 2단계 — 형식별로 정리

카피는 쓰이는 자리마다 형식이 다르다. 앵글 후보를 아래 3형식으로 정리한다.

- **헤드라인**: 시선을 붙잡는 한 방. 짧고 강하게. 보통 한 줄.
- **슬로건**: 브랜드를 오래 대표하는 문구. 외우기 쉽고 반복해도 안 질리게.
- **CTA (행동 유도 문구)**: "지금 사세요"처럼 다음 행동을 부르는 말. 동사로 끝나고 구체적으로.

### 3단계 — 브랜드 톤 체크

각 후보가 브랜드 톤을 지키는지 확인한다. 멋있어도 톤에서 벗어나면 탈락이다. (진중한 브랜드에 유행어 카피, 존댓말 브랜드에 반말 등.) 금지어·과장 표현·법적 문제 소지(허위·비교광고)도 이때 거른다. 톤을 벗어난 후보는 버리거나 톤에 맞게 고쳐 쓴다.

> 완성 카피의 최종 검수·반려는 별도 역할(creative-direction)이 한다. 여기서는 **자기검열 수준**의 1차 톤 체크만 한다.

### 4단계 — A/B 변형 제공

고른 후보에 대해 **비교 테스트용 변형**을 만든다. 실무는 하나를 확정하지 않고 두세 개를 돌려 반응을 본다.

- 변형은 **한 가지 변수만** 바꾼다. (예: A는 질문형, B는 단정형 — 나머지는 동일.) 여러 개를 동시에 바꾸면 뭐가 효과를 냈는지 모른다.
- 각 변형에 "무엇을 테스트하는지" 한 줄 메모를 단다.

## 산출물 형식

```markdown
# [제품/캠페인] 카피
> 타깃: [ ] / 담은 인사이트: [ ] / 브랜드 톤: [ ]

## 감성 앵글
- 헤드라인: "..."
- 헤드라인: "..."

## 기능 앵글
- 헤드라인: "..."

## 유머 앵글
- 헤드라인: "..."

## 슬로건 후보
- "..."

## CTA 후보
- "..."

## A/B 테스트 세트 (추천안)
- A: "..."  ← 테스트 포인트: [질문형]
- B: "..."  ← 테스트 포인트: [단정형]

## 추천 & 이유 (2~3줄)
```

## 품질 기준

- [ ] 감성·기능·유머 3앵글 모두 시도
- [ ] 헤드라인·슬로건·CTA 형식 구분
- [ ] 모든 후보가 브랜드 톤 준수(벗어난 것 없음)
- [ ] A/B 변형은 한 변수만 다름 + 테스트 포인트 명시
- [ ] 담기로 한 인사이트가 실제 카피에 반영됨
- [ ] 과장·허위·비교광고 소지 없음
- [ ] CTA는 동사로 끝나고 구체적

## 자주 하는 실수

- **한 앵글만** → 더 좋은 각도를 놓친다. 3앵글 필수.
- **제품 설명을 카피로 착각** → "무설탕 200ml"는 스펙이지 카피가 아니다. 마음을 건드려야 한다.
- **멋만 있고 톤 무시** → 브랜드와 안 맞으면 탈락. 3단계에서 거른다.
- **A/B에서 여러 변수 동시 변경** → 원인을 못 가린다. 한 번에 하나만.
- **유머가 선을 넘음** → 재미있으려다 브랜드를 다치게 한다. 자기검열.
- **CTA가 흐릿함** → "많은 관심 부탁"보다 "지금 첫 구매 20% 받기"처럼 구체 동사.

## 재호출 지침

- "카피 다시" → 같은 재료로 새 후보를 발산한다(앞 결과와 겹치지 않게).
- "더 감성적으로/유머로" → 해당 앵글을 집중 강화해 재작성한다.
- "톤 수정" → 브랜드 톤 기준을 다시 적용해 벗어난 후보를 손본다.
- "헤드라인만/CTA만 보완" → 해당 형식만 다시 뽑고 나머지는 유지한다.

> AI는 매번 다른 카피를 낸다. 그게 장점이니 여러 번 돌려 후보를 늘려도 좋다. 다만 최종 선택과 톤 판정은 사람 또는 검수 역할이 한 번 더 본다.
